# Long Tail project charts

This is a sub project in the UnicornEng-Diagrams repo.
This is a dev container set up used to run a Jupyter notebooks that produce charts to visual the load of long tail project.

## How to use

- Open this foler in VS Code as a Dev Container.
- Open Jupyter notebook server on <http://localhost:8889/lab/>
- See the notebook as _/work/plots_longtails.ipynb_
