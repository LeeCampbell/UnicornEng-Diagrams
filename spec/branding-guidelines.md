# Branding Guidelines

Visual identity system for Unicorn Engineer diagrams.

## Color Palette

### Primary Blues (4-tier hierarchy)

Darker shades indicate higher importance or emphasis.

| Tier        | Hex       | Usage                                       |
|-------------|-----------|---------------------------------------------|
| Primary     | `#4477aa` | Borders, strokes, headings, key fills       |
| Secondary   | `#7db8da` | Primary fills, medium emphasis              |
| Tertiary    | `#b6d7ea` | Secondary fills, light emphasis             |
| Quaternary  | `#d4e6f1` | Background fills, lowest emphasis           |

### Grays

| Hex       | Usage                              |
|-----------|------------------------------------|
| `#343a40` | Dark text, chart titles            |
| `#495057` | Medium text, axis labels           |
| `#6c757d` | Annotations, secondary text        |
| `#f8f9fa` | Page/chart backgrounds             |
| `#dee2e6` | Chart area borders, subtle dividers|

### Accents

| Hex       | Usage                              |
|-----------|------------------------------------|
| `#dc3545` | Error, warnings, percentile lines  |
| `#00b894` | Success, positive indicators       |

## Typography

- **Font family:** `Arial, sans-serif` (all text)
- **Title:** 18-20px, bold
- **Axis/section labels:** 14px, bold
- **Quadrant/node labels:** 15-16px, bold
- **Body text:** 12px
- **Small text:** 11px

## Design Principles

- **Clarity first** — No decorative elements. Every visual element serves a purpose.
- **Hierarchical color gradation** — Darker blue = more important. The 4-tier blue system encodes emphasis through shade.
- **Minimalist** — White or `#f8f9fa` backgrounds, clean lines, generous whitespace.
- **Consistent structure** — All diagrams follow the same color, font, and spacing conventions for a unified look across the book.
